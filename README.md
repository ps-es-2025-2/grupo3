# MindSystem

Sistema de orientação vocacional e acompanhamento psicológico desenvolvido como projeto acadêmico da disciplina de Engenharia de Software.

## Sobre o Projeto

O MindSystem é uma solução voltada para auxiliar no processo de orientação vocacional e oferecer suporte psicológico através de uma plataforma digital integrada. O projeto foi desenvolvido seguindo metodologias de engenharia de software, com documentação completa das fases de análise, design e implementação.
